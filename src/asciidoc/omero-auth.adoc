== OMERO auth sequence

----
  +---------------------+
  + Blitz               +
  +---------------------+
  | PermissionsVerifier |
  | #checkPermisssions  |
  +---------------------+
    :   +--------------------------+
    |   | SessionManager           |
    +-> | #executeCheckPassword    |
        |  #executeCheckPasswordRO |
        |  #executeCheckPasswordRW |
        +--------------------------+
          :   +----------------------------+
          |   | AdminService               |
          +-> | #checkPassword(usr,pwd,ro) |
              +----------------------------+
                :   +----------------------------+
                |   | PasswordProvider           |
                +-> | #checkPassword(usr,pwd,ro) |
                    +----------------------------+
----
